# R.tournee

Projet fullstack:
- `frontend`: React + Vite (UI)
- `backend-next`: Next.js (API)
- `backend` (NestJS) : ancien essai, plus utilisé pour l'instant

## Prérequis
- Node.js 20+
- npm 10+

## Lancer le frontend
```bash
cd frontend
npm run dev
```

## Lancer le backend Next.js
```bash
cd backend-next
npm run dev
```

## Build
### Frontend
```bash
cd frontend
npm run build
```

### Backend Next.js
```bash
cd backend-next
npm run build
```
